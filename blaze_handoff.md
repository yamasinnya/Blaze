# Blaze Roguelike — Claude Code引継ぎファイル

## プロジェクト概要

MTGのリアルパックを使ったソロローグライト。
デジタル側はゲーム進行・敵AI・ライフ管理のみ担当。
カードの処理はプレイヤーの自己申告ベース。

**想定ユーザー：** yuki（行弘賢、MTGプロプレイヤー）へのプロツアー優勝祝いとして制作。
**配信映えを意識したUX。**

## ゲームルール

### デッキ構成（暫定、実地調整中）
- 呪文11枚（パックから土地・広告除いた全カード）
- 基本土地5枚（各色1枚、対戦間に自由入れ替え可）
- 特殊土地4枚（デジタル管理。基本色＋ダメージで任意色）
- 合計20枚

### 基本設定
- 初期手札：7枚
- 自分のライフ：40点
- 毎戦闘後：5点回復（初期レリック）
- 相手ライフ：10点（ステージ1）

### パック運用（ローグライト構造）
1. 1パック目開封→デッキ構築→戦闘
2. 勝利→2パック目開封→裏向きシャッフル→数枚提示→1枚選んでデッキ交換
3. 繰り返し。ボス撃破でレリック獲得

---

## 戦闘フロー

```
自分のターン（緑表示）
　↓
ターン終了ボタン
　↓
①×マークのライオンを削除
②召喚酔い解除（前ターン以前に出たやつ）
③召喚フェイズ（間隔の倍数ターンなら召喚、召喚酔い状態で登場）
　↓
一拍（500ms）
　↓
アタッカーがいれば → 相手ターン（赤表示）+ アタック宣言中バッジ
　↓
ライオンをクリックして状態変更（いつでも可能）
  通常 → 除去(×赤バッジ) → ブリンク(半透明Bバッジ) → 通常
　↓
「戦闘解決」ボタン
  - ×：削除
  - ブリンク：ダメージなし
  - 通常：ダメージあり
　↓
自分のターンに戻る
```

## ステージ1：敵スクリプト

### 雑魚：サバンナライオン無限召喚
- 召喚間隔：2の倍数ターン（コントロールパネルで調整可）
- 召喚数：3ターンごとに+1体（調整可）
- 攻撃力：2点/体（調整可）
- **速攻なし**：召喚ターンは攻撃しない、次のターンから攻撃

### エリート：筋肉スリヴァー（未実装）
- 毎3ターンに1体追加
- スリヴァー全体+1/+1ロード効果
- 実質3/3が増え続ける

### ボス：初祖牙、アラーボ＋サバンナライオン（未実装）
- 1ターン目：アラーボ召喚（2/2、ロード効果）
- 3ターン目以降：3ターンごとにライオン召喚（アラーボ効果で3/2）
- ライオン召喚のたびに1/1猫トークン生成
- アラーボ先に倒せばトークン生成停止

```

技術方針：
- Phaser.js + 素のJS、フレームワークなし
- Node.js不要、CDN読み込みのみ
- 自分で読めて直せるシンプルなコード
- ファイル構成はenemies.js / relics.js / map.jsに分割予定
```
